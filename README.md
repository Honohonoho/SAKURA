# SAKURA 一个 VUE UI 组件

[![Build Status](https://travis-ci.org/Honohonoho/SAKURA.svg?branch=master)](https://travis-ci.org/Honohonoho/SAKURA)

## 介绍

SAKURA(樱花) UI 是一个好用的 UI 组件库，提供了常用的 UI 组件， 适合移动端和 PC 端使用。

## 开始使用

1.安装

```
npm install sakura_ui -S
```
